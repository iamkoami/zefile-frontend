# Metropolis Font

Place the Metropolis font files in this directory:

- `Metropolis-Regular.woff2` (weight: 400)
- `Metropolis-Medium.woff2` (weight: 500)
- `Metropolis-SemiBold.woff2` (weight: 600)
- `Metropolis-Bold.woff2` (weight: 700)

These fonts are configured in `/app/layout.tsx` and will be used throughout the application.

## Font Weights Used

- **Regular (400)**: Body text
- **Medium (500)**: Emphasized text
- **SemiBold (600)**: Headings
- **Bold (700)**: Strong emphasis

## Temporary Fallback

Until the font files are added, the application will use system-ui and arial as fallbacks.
